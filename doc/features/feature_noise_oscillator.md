# Noise Oscillator Feature

We can have a noise oscillator that can inject additonal noise into the incoming signal.
Said noise can also be used to test drive the dsp effects in the standalone app.
As such it can be router into the flow from the beginning or at various places in the flow.
There are different types of noise, white, pink, and possibly FM noise. A special knob FM Mod can be used to throw that signal into the filters and possibly via the mod matrix.
