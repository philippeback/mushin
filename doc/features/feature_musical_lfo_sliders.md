# Musical LFO Sliders

The LFO1 and LFO2 sliders are meant to be used to give musical resuls.
As such, they also support the shift dragging etc like the knows so that a precise value can be set.
When the LFO value is clicked one can input a value in place using the keyboard up to 2 decimal places.
No negative value allowed.
Also the range is not linear. There is a lot of value in being able to dial in lower values.
Maybe we have a range selector or we have a third of the knob range for low freq LFO, one third for mid lfo freq and one third for higher LFO freq.
The point is to be able to modulate easily.
Maximum modulation frequency to be defined, but 20Hz already big.

# LFO shapes

The LFO shape is currently selected in a dropdown.
Given that there are currently 5 options, this is ripe for using a pictogram for each (can start by being some letters but pictogram of wave nicer)
So that set of 5 pictograms acts like a dropdown.
The selected item is glowing in the primary color.
