# Delay feature

This is an FX that can be enabled in the signal chain
There is an enable button somewhere in an FX panel.

The effect has a send knob that helps in dialing it.

The effect is a special kind of delay that has a set of definie algos.
Each algo is selected by a knob that is not continuous but discrete.
Each notch is selecting an algorithm.

Algorithms are:

* Dub Delay
* Diffuse Delay
* Ethereal Echo Delay
* Kartman-Strong Delay
* Multi head with drift delay

Each of the delays provide a rich and distinct out of the box experience.
This to be specified as we work through the feature.

Delays are tempo synched (several options do exist in a dropdown )
