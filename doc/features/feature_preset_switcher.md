# Feature: Preset switcher

There are two arrows (left and right) near the presets dropdown.
When clicked, the arrow selects the previous or the next preset of the list.
When start of the list is reached it goes to the last preset in a round robin fashion
Same boat for the last preset and next arrow.

There is a small search box just near the presets box that allows for dynamic filtering of the preset.
When text is input, the preset list is filtered dynamically.
If there is nothing found, previous and next arrows have no effect.
