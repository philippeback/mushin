# feature suboscillator

A suboscillator aligned with the signal can be activated.
The suboscillator can be of various wave forms like square / PWM / SAW and be doing some stuff in the mod matrix (TBD)
The suboscillator can be tuned to musically useful intervals vs the input processed signal.
The point is to beef us said signal. 
The sub oscillator is in mono.
There is a glowing led light activating when the sub is running and aligned on the sub "intensity"
