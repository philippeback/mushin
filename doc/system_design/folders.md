The folders in the project root are:

- `.agent`: Contains Gemini-specific configuration and project context.
- `.cache`: Local cache directory (likely for `clangd`).
- `.gemini`: Gemini CLI internal configuration and skills.
- `.vscode`: Visual Studio Code settings and tasks.
- `build`: Build artifacts (Ninja-based).
- `build2`: Dedicated development build directory (MSBuild).
- `doc`: Project documentation.
- `skills`: Custom skills for the environment.
- `Source`: C++ source code for the plugin and web assets.
- `specs`: Project specifications and overviews.
